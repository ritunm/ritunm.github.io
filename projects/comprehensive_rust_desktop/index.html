Building

Mac

  • Install npm and electron
  • Build with electron Builder
npm install -g electron
npm install electron-builder --save-dev
npm run build

Windows

  • Install npm and electron
  • Build with electron Builder
npm install -g electron
npm install electron-builder --save-dev
npm run build

Windows

Open Programs and Features from Control Panel
